A complete Flutter + Firebase integration project demonstrating authentication, real-time database usage, and clean architecture implementation.

This repository shows how to connect Flutter apps with Firebase services in a production-ready way.

 # 🚀 Features

🔐 Firebase Email & Password Authentication

🔵 Google Sign-In Authentication

🔄 Real-time Data Sync

☁️ Cloud Firestore Integration

📦 Clean MVVM Architecture

⚡ Error Handling & Validation

 🔥 Secure Firebase Configuration
